# Импорт данных

## Импортирование данных заявки с сайта orgeo.ru

Текущая версия по умолчанию поддерживает CSV форматы.
Рекомендуется использовать "Excel CSV (кодировка Windows-1251)".

`Файл -> Импорт -> Orgeo CSV`
